# Grapes Disease Detection using Machine Learning & Deep Learning

This project is an AI-powered Grapes Disease Detection system developed using Machine Learning (ML), Deep Learning (DL), TensorFlow Lite, and Flutter. The application detects diseases in grape leaves using image classification models and provides real-time predictions directly on mobile devices.

The project combines deep learning-based image processing with Flutter mobile application development to create a fast, lightweight, and user-friendly solution for smart agriculture.

---

## Project Overview

The system is designed to:

- Detect grape leaf diseases from images
- Use Deep Learning models trained on grape disease datasets
- Convert trained models into TensorFlow Lite (`.tflite`) format
- Integrate the models into a Flutter mobile application
- Perform real-time on-device inference without requiring internet access

---

## Diseases Detected

The model is trained to classify grape leaf images into different categories such as:

- Black Rot
- Esca (Black Measles)
- Leaf Blight
- Healthy Leaves

---

## Flutter & TensorFlow Lite Integration

The trained deep learning model is converted into TensorFlow Lite format and integrated into the Flutter application using the `tflite_flutter` plugin.

### Features of Integration

- Real-time disease prediction
- Mobile-friendly lightweight model
- Fast image inference
- Offline functionality
- Cross-platform support (Android/iOS)

---

## ⚙️ Workflow

1. Collect grape leaf image dataset
2. Train CNN/Deep Learning model
3. Evaluate model accuracy
4. Convert model to `.tflite`
5. Integrate model into Flutter application
6. Capture or upload leaf image
7. Predict disease using on-device inference
